# Usage

<!-- usage -->

```sh-session
$ npm install -g azez
$ azez COMMAND
running command...
$ azez (-v|--version|version)
azez/0.0.0 darwin-x64 node-v10.15.0
$ azez --help [COMMAND]
USAGE
  $ azez COMMAND
...
```
